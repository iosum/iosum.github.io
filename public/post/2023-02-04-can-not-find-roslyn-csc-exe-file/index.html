找不到 bin\roslyn\csc.exe 檔案

By Casey | February 04, 2023 | 1 minutes
ASP.NET

錯誤訊息

找不到路徑 ‘D:\xxx\bin\roslyn\csc.exe’ 的一部分。

解決方法

找到 packages\Microsoft.CodeDom.Providers.DotNetCompilerPlatform.2.0.0

進入 tools

進入 Roslyn45

貼到專案的 Bin folder，並重新命名為 roslyn

再重啟專案，即可成功
